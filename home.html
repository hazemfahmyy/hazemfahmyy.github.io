Safety Analysis of Image-processing DNNs via Clustering-based Approaches

Abstract: This work introduces robust white-box and black-box safety cluster architectures to parse latent execution spaces within deep vision convolutional structures. By implementing dimension reduction alongside unsupervised feature isolators, the framework automatically identifies operational vulnerability pockets, enabling focused retraining blocks that minimize regression risks.

HUDD: A Tool to Debug DNNs for Safety Analysis

Abstract: Demonstrates HUDD (Heatmap-based Unsupervised Deep Debugging), an automated software engineering utility tracking activation maps to identify error root-causes. By clustering uniform activation patterns, the framework isolates systematic model tracking failures without human intervention with 93% verified purity.
